CoachIQ is a modern desktop application that helps coaches and program managers analyze Excel spreadsheets using AI.

## Features

-  Upload Excel spreadsheets
-  Ask questions in natural language
-  Compare seasons and trends
-  Generate PDF reports
-  Interactive dashboards
-  Native desktop application
-  Local-first data processing

## Planned Features

- AI-powered insights
- Multi-year trend analysis
- Revenue analytics
- Enrollment forecasting
- Returning player analysis
- Dark mode
- Automatic updates

## Built With

- Python
- PySide6
- pandas
- SQLite
- Plotly
